== ðŸ“œ Lab 2: Writing and Running a Playbook

=== ðŸ”¹ 2.1 Create the Playbook File

Create a file named `mca_demo.yml`:

[source,yaml]
----
- name: MCA Ansible Playbook Demo
  hosts: web
  tasks:
    - name: Create a welcome file
      copy:
        content: "Welcome MCA Students to Ansible Lab!\n"
        dest: /tmp/welcome.txt

    - name: Ensure a directory exists
      file:
        path: /opt/mca-lab
        state: directory

    - name: Install tree (optional)
      package:
        name: tree
        state: present
----

=== ðŸ”¹ 2.2 Run the Playbook

[source,bash]
----

----

=== ðŸ”¹ 2.3 Verify the Changes

Log in to any managed node and run:

----
